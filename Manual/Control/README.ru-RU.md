# Управление

Ярость Песков использует клавиатуру и мышь в качестве устройства ввода

## Выход

- Esc

## Пауза

- P

## Открыть Меню

- Левый Ctrl
- Правый Ctrl

## Передвижение и Взаимодействие

Игра включает такой набор комбинаций управления с клавиатуры:

Обычная
- W-A-S-D + E
- ^-&lt;-v-&gt; + Space/Enter

Числовая
- ^-&lt;-v-&gt; + Space
