# Controls

Desert Rage use keyboard and mouse as user input

## Pause

- Esc

## Exit

- Shift + Esc

### /!\ Warning

When you quit the game, your progress will be truncated till last save.

## Open Menu

- Left Ctrl
- Right Ctrl

## Movement and Interaction

Game include such popular keyboard combinations as:

Usual
- W-A-S-D + E
- ^-&lt;-v-&gt; + Space/Enter

Num Pad
- ^-&lt;-v-&gt; + Space
